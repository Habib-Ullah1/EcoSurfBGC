# EcoSurfBGC

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19565129.svg)](https://doi.org/10.5281/zenodo.19565129)

This repository contains R, Python, and shell scripts for the ecosystem-resolved biosynthetic gene cluster (BGC) atlas described in Ullah et al. (2026).

## Contents
- `scripts/*.R` — R scripts for data processing, visualization, and figure generation
- `scripts/*.py` — Python scripts for parsing antiSMASH/Big-SCAPE outputs and automation
- `scripts/*.sh` — Shell scripts for genome downloads, antiSMASH runs, and workflow orchestration
